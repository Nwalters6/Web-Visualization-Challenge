Summary: Weather Plots



Visualizations



The purpose of this project was to analyze how weather changes as you get closer to the equator. To do this we first pulled data from the OpenWeatherMap API to assemble a dataset of over 500 cities.


After assembling the dataset, Matplotlib was used to plot various aspects of weather vs. latitude. The factors analyzed included: temperature, cloudiness, wind-speed and humidity.


Matplotlib was also used to plot aspects of the weather using longitude vs. latitude. The intent was to visualize a map using weather plots. This site provides the source data and visualizations created as part of the analysis, and the explanations and descriptions of any trends and correlations.
